# Version 3.0.0 - 代号「门」【Gate】

---

这次是我。

## 第三个站

esap.ltd 上线了。

ESAP Ltd 创意科技工作室的官方网站。首页、关于、产品、联系。两种语言。Typewriter 打字效果，ParallaxSection 视差背景，CountUp 数字动画。团队页面展示 1547、1548、1549，圆形头像，品牌色边框。

Apache 2.0 许可证。支持无 JavaScript 环境。

## 角色页面重构

weare-website 的角色图片系统重构。从单一图片改为三层结构：`avatar`（头像）、`hero`（背景图）、`logo`（标志）。九个角色全部更新，每个角色一个文件夹。

角色详情页加了 ArrowLeft 返回图标。Icon 组件补了一批新图标。

## 赞助商和加载

story-website 赞助商系统支持多图展示。关于页面加了骨架屏，`animate-pulse` 加载动画。

## 共享组件

shared-ui 新增三个组件：CountUp（数字计数动画）、PageTransition（页面过渡）、ParallaxSection（视差滚动）。都支持无障碍模式。

## 依赖更新

framer-motion 12.35.2，Biome 2.4.6，Tailwind CSS 4.2.1。类型定义改为精确版本。

## 门

2.0 是 story.esaps.net。3.0 是 esap.ltd。

weare 是角色档案，story 是故事阅读，esap.ltd 是对外的门面。

三个站，三个入口。
