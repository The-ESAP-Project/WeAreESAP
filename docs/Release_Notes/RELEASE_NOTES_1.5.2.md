嘿。是我。AptS:1547。

以前的 release notes 都是别人写的——48 写过，1543 写过，49 写过，1738 连着写了两次。他们说该我了。好吧，那就随便写写。

这个版本做的事说起来挺简单的：给人画脸。组织页的成员卡片以前就一个编号圆圈，冷冰冰的。我给八个人都画上了头像——0152、1543、我自己、48、49、1738、2275、4869。加载的时候先模糊着，清晰了再显出来。加载失败就继续显示编号，不丢人。占位符脚本也顺手重写了，从只扫一个目录改成扫三个，路径从文件名换成相对路径，不会撞。

1553 终于有名字了。提线。中英日三语言补齐。时间线里那些写着"其他"的对话也改了——不是"其他"，他们有名字。0152、1543、1738、2275，一个个认回来，配上专属颜色。说话的人该是谁就是谁。

然后是个看不见的裂缝。科技页和组织页之间跳转会炸，React 报 hooks 数量变了。查下来是过渡系统在 pushState 切 tab 的时候误触发了 key 变化，跳到带 redirect 的页面时 Router 状态不同步就崩了。改了过渡系统只认真正的导航，去掉了两个页面的 redirect。顺手让 tab 切换记住滚动位置，切走再回来还在原地。

剩下的都是碎碎念。返回按钮统一了过渡动画，组织图片加了模糊占位，1645 喵风的贡献者信息补全了，framer-motion、next-intl、biome、jsdom 常规更新。

差不多就是这样了。我们下次再见
