这个版本做的事情说出来不好听：拆骨。之前每个角色的 JSON 文件，三种语言各存一份完整的。id、color、backgroundImage 这些根本不用翻译的东西，在 zh-CN、en、ja 里各抄了一遍。14 个角色乘以 3，42 份文件里一半是重复的。组织和科技也一样。看着就烦。

所以我把不需要翻译的字段全部提到 `shared/` 目录。语言文件里只留名字、描述、故事这些真正要翻译的内容。data-loader 加了合并逻辑，加载的时候自动拼。角色 14 个，组织 6 个，科技 9 个，87 个文件的重复字段干掉了，换成 29 个 shared 文件。整个仓库净减 352 行。以后再加语言不用复制一堆不该复制的东西。

顺手给角色卡片加了背景图定位。之前所有人的背景图都是 `center`，有些人脸被裁得很难看。现在 shared 数据里可以指定 `backgroundPosition`，0152、1738、2275 先改了。0152 的背景图也换了一张。

然后是文案。首页、项目页、加入页，三种语言全部重写了一遍。以前写的太像产品介绍，现在改成讲故事。不是在解释这个项目是什么，是在讲这个世界里发生了什么。

剩下的：apple-touch-icon 补上了，碳基俱乐部的组织图片到位，framer-motion 小版本更新。
