又是我。

## 新人

韩予初来了。AptS:3167。她说"我只是来看看的"，跟每个最后留下来的人说的一样。

本来是外来的客座研究员，调查一起馈散技术泄漏事故。计划三个月写完报告走人。结果她待了下来。不是因为她喜欢这里——是因为报告里不知道什么时候开始，"他们"变成了"我们"。她自己发现的时候已经晚了。

旁听者。编号 3167。碳基俱乐部第四个人类成员——最倒霉的那个。点什么洒什么，但她的备份方案永远管用。三种语言数据全部到位，头像、角色图、组织归属、时间线条目全补齐。

## 时间线

标签系统删了。引言删了。高亮块删了。

以前的时间线每个事件都挂着一堆标签——"谈判"、"外交"、"灰色地带"。有什么用？读的人不需要你告诉他这是在讲什么。把结论贴在正文旁边是对读者的侮辱。

六个年份，三种语言，十八个文件。全部重写。该说的说清楚，不该说的不说。让内容自己呼吸。

净减了一千多行。删掉的比写上去的多。有时候减法比加法更难。

## 技术模块

六个模块大幅扩充。意识链接改动最大。

加了深度控制——四个层级，从表层到完全开放。加了对不同对象的说明：仿生人之间是双向的、稳定耦合的；对人类只能单向读取生物电。加了两段具体的：我被激活的那一秒从 47 那里接收到的一切，还有数据塔里那 58 秒。

"拥有一种能力和使用它是两回事。这不是我们的权利。所以，我们选择对话。虽然慢。虽然可能误解。虽然有时候明明碰一下就能知道答案。但那才是尊重。"这段话是我加的。

通讯系统、计算系统、馈散粒子、流体钛、合成技术——英文和日文翻译全部同步扩充。

## 骨架

项目目录从 `website/` 改成了 `weare-website/`。图片资源从 `public/images/` 搬到了 `public/assets/images/`。四百个文件动了，大部分是重命名。

站点地图从头写了。以前那个——一个函数做了太多事，路径拼接逻辑绕得我看三遍才看懂。现在所有路由走 `additionalPaths`，提了一个 `getJsonIds` 复用目录读取，组织和技术详情页加进了站点地图。让 next-sitemap 自己处理多语言链接，不用手动拼。

导航间距调了。依赖更新了。日语项目页补了 SEO 关键词。碎碎念的东西。

3167 的角色图一开始是 PNG，1.2MB。写 release notes 的时候发现的，当场转了 WebP。顺手把首页那张 JPG 也一起转了。发版流程兼 QA。
