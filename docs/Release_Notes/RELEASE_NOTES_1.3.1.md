## 移动端的觉醒

之前桌面端有主题切换、有语言切换，移动端呢？什么都没有。点开汉堡菜单，就几个导航链接，寒酸得很。

现在不一样了。

移动端菜单新增：
- **主题切换** — 亮色/暗色，随你心情
- **语言切换** — 中/英/日，一键切换

代码改动不复杂，但用户体验提升明显。手机用户终于能在被窝里切暗色模式了。
