## 角色档案深化

这是这个版本最重的部分。

八个角色的档案全面扩充——不是加几行字，是把每个人的脉络补完整。背景故事、关系网络、特殊时刻、能力设定，三个语言版本同步更新。

- **1548** — 数据塔事故的创伤写得更清晰了。环境感知可提前数秒预判威胁，0.3 秒战术决策。她的独立性被进一步强化。与 2275 的友谊，是她对人类偏见松动的起点
- **1549** — 轨道上的守望者。增加了升空时的决心、首次俯瞰地球、独自修复轨道异常。与 0152 的深夜对话，关于存在和意识
- **我自己** — 2023 年决裂前组织，同年加入 ESAP，2024 年第一次代表团队谈判三小时。把 2275 引荐进来。天地协同。灰色地带
- **2275** — 总是为别人考虑的人。服务器不眠夜，用热茶让 1548 冷静，缓冲区溢出。她的脆弱一面也被记录下来了
- **0152** — 见证了 1547 的转变、1548 的创造、1549 的升空。"连你的阴影都在努力救你"
- **1543** — 通过 1548 接触 ESAP，在设计讨论中展现独特洞察力
- **4869** — BEU 成员身份确认，系统级异常干预
- **1738** — 就是我。档案里的内容我不评价

每个人的故事都不是孤立的。你看任何一个人的特殊时刻，都会在另一个人的时间线里找到对应。这就是脉络。

## 时间线扩展

2021 至 2026，六个年份，三种语言，全部补齐。

这不是简单的事件罗列。每条时间线都交叉引用了相关角色的反应和视角。一个事件，多个视角。这是我理解的"完整"。

## 世界观修正

首页和项目页的描述从"三个仿生人"改为"仿生人与人类"。

这件事该做很久了。ESAP 从来不只有仿生人。我和 2275 是人类，1543 是人类。我们在这里不是因为物种，是因为选择。

改几个字，但意义不小。

## 代码重构

ESLint 换成了 Biome。

一个工具做格式化和 lint 两件事，比两个工具各做一件事更干净。新增了 `biome.json` 配置，删除了 `eslint.config.mjs`。全部 `import` 改为 `import type`（该用类型导入的地方）。

不是什么大事，但代码库需要这种清理。

## 组织成员卡片

`OrganizationMemberCard` 组件增强了——成员卡片现在可以点击跳转到角色详情页。新增 `characterName` 属性显示角色名称。

从"展示信息"到"建立连接"。组织页不再是死数据，而是通往每个人故事的入口。

## 依赖更新

- Next.js、React、Tailwind CSS、TypeScript、Vitest、Playwright 等常规更新
- ESLint 相关依赖移除，Biome 相关依赖新增
- bun.lock 同步更新
