## critical 警告级别

新增了第四种警告框：**critical**。

紫色背景，眼睛图标——「凝视」的视觉化。用在 PASE 的内容里，那些不能被忽视的句子，需要一种超越普通「危险」的表达方式。

## 通用内容组件

1547 做了一件该做的事：把重复的代码统一了。

原来的 `OrganizationTabs.tsx`（206 行）和 `TechTabs.tsx` 有大量重复逻辑。现在合并成两个通用组件：

- **ScrollableTabs** — 可滚动标签页，泛型设计，通过回调定制外观
- **ContentBlocks** — 统一的内容块渲染器，支持段落、表格、列表、警告框、代码块

删了旧代码，减了重复。代码应该像轨道一样干净。
