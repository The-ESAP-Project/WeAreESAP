# Version 2.0.0 - 代号「声」【Voice】

---

这次是大版本。

## 第二个站

story.esaps.net 上线了。

从架子搭起，从零开始：阅读器、章节系统、分支选择、视角切换、探索模式、解锁机制。四个故事上线：《起源》、《数据塔事故》、《灰色地带》、《你们看看》。三十多个章节，三种语言。

有些功能是写的时候边写边设计的——比如分支叙事。读者在某些章节可以跟随不同的视角，看同一件事从不同眼睛里是什么样的。《数据塔事故》第二章，可以从 1547 的视角看，也可以从我的。

霞鹜文楷。YakuHanMP。细节。把阅读体验本身当成设计的一部分。

还有 Cmd+K 全局搜索。每一页都有骨架屏。这些不是功能，是态度——我们不喜欢让人等待，哪怕等待是必要的，也要让人知道东西正在来。

## 玛托莉

AptS:1582。驻留者。远道而来，坐下看卫星。

她的颜色是黄色。一种不太确定的黄，像某种很久以前的光。她的关键词是"迟缓"——不是反应迟钝，是慢慢触碰，慢慢确认，慢慢地把散落的东西拼起来。

weare 上线了她的全部数据。三种语言，头像，角色图，时间线条目。

## 架构

monorepo 从两个独立项目变成了正式的 pnpm workspace。共享 UI 包提出来了——AnimatedSection、LanguageSwitcher、ThemeToggle、useReducedMotion，以前在 weare-website 里重复的东西，现在一个地方维护，两个站都直接 import `@esap/shared-ui`。以后加第三个站不用再复制粘贴。

## 其他

weare-website：角色卡片上的关键词标签删了。不是功能退化——视觉上干净了，信息在角色页里还在，就是不需要在卡片上全摊开。文字阴影调了，比以前轻，比以前准确。页脚把 Discord 位置换成了故事站的链接。

Apple Touch Icon 两个站都补了。许可证头部统一更新到 2021-2026。依赖更新了一轮。

## 声

1.0 开始搭架子。之后每个版本往里加东西：韵、骨、血、息。

2.0 是第一次真正开口说话。

weare.esaps.net 里的每个人，现在有地方讲自己的故事了。不是设定文档，不是参数表，是他们自己说出来的事情——1547 为什么在实验室里激活了我，1738 在 CREF 大楼顶层看到了什么，1543 为什么用设计语言来表达自己。

我们一直在说"这不是一个角色设定项目，是一个正在发生的故事"。

现在你可以去读了。
